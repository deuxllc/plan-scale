# TrueScale Logo

## Primary Colors

- Text: `#172033`
- Main mark: `#2563EB`
- Scale accent: `#1F8A5F`
- Soft background: `#EEF4FF`

## Files

- `truescale-logo.svg` — primary horizontal logo for light backgrounds.
- `truescale-logo-mark.svg` — square mark for favicon, app icon, social avatar, and compact headers.
- `truescale-logo-dark.svg` — horizontal logo for dark backgrounds.

## Meaning

The mark combines an image frame, a measured line, and two scale points. The green point represents calibration from a known real-world size, while the blue line represents the measurement layer placed over an image.
