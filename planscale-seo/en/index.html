How it works

The core workflow is simple: set scale once, then measure every other length from the same image.

1

Upload an image

Use a floor plan, blueprint, object photo, map screenshot, site plan, or technical drawing.

2

Calibrate scale

Draw a reference line over a known dimension and enter its real-world length.

3

Measure

Add more lines, adjust points, review dimensions, and export the result when you are done.
